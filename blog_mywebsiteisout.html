My website is out! Or how to do a website without very little web design knowledge

23.10.23

Oh yes: my website is out! I've been thinking for a while whether I should actually have it or not. I remember a few months ago there had been a long conversation on X between some economists on the importance of personal websites for emerging scholars. Well, assuming that they are important for academics, is it the same for people working in the grey area of policy X research X else? I don't know. Still, it was fun to build this website. We'll see in the coming months if it is actually useful or not.

How did I make it? I should be honest: it was a looong process. Not really long to build in itself. But definitely very long to obtain what I wanted. This website is the third attempt I make in a few weeks. I created the first one with Google Sites. Pros: very easy to make and quite aestetically customizable. Cons: not much fun in the making and not (so) much freedom. Then I found this YouTube video that inspired me to build from scratch with HTML + CSS + JS. So I decided to go this way. The second attempt I made was using a beautiful CSS template that would have made my page look like Notion (here). I spent long long time customizing it all. But then I realized I didn't really like it. So, I went for a third attempt. With some more precise inspiration, some stealing here and there from layouts available online, and a lot (really a lot) of patience, I managed to reach this result. I'm pretty satified. I hope you appreciate it as well.

What will I do here? Well, it is going to be mainly a landing page. But I'd be happy to start populating it with some of the things I write. It could be notes on my projects, new ideas, reflections to make my mind clearer. We'll see. But I am happy to share what I learn.
